# Smart Waste Sorting System
 An Object Recognisation system to differentiate waste in multiple types, currently it has only Biodegradable and NonBiodegradable as options

# Features
- Object Detection
- Classification Between Biodegradabe and Non-Biodegradable
- Ability to take actions according to Decision Made

# Tech Stack
- OpenCV
- LiteRT(Formally known as TFLite)
- Python
- NumPy

# Hardware
- Raspberry PI 5
- 16-Channel PWM Servo Driver
- Servo Motors X 2
- IR Obstacle Sensor
- IR Webcam/Camera Module

### Note: This Project needs 12C Interface and Camera Support Enabled in Raspberry PI 5 and it is mandatory to install LiteRT and OpenCV along with Python 3.13.* to use the following code.

### ⚠️ WARNING: This code is only tested on Raspberry PI 5 and Hobby Servo Motors, and this code is mostly Static coded, so it may not work as expected.
